# メニュー抽選Webアプリケーション

Reactを用いて制作した、日常の意思決定を支援するWebアプリケーションです。

## 概要
「今日のご飯が決まらない」という日常の課題をもとに、条件を選択することで料理をランダムに提案するWebアプリを作成しました。

## 機能
- 料理ジャンル・時間帯の選択
- 条件に応じたランダム表示機能
- 過去の選択履歴表示機能

## 使用技術
- React
- JavaScript
- HTML / CSS
